Welcome to FastMediaSorter!

1. Application Goals

FastMediaSorter is designed for rapid sorting and management of large collections of photos and videos. The primary goal is to provide a highly efficient workflow for users who need to organize media files into different folders quickly, using extensive keyboard shortcuts to minimize mouse usage.

2. Full List of Functionality

File Viewing & Navigation

  • Next/Previous File: Navigate with →/← arrows, PgDn/PgUp, N/P keys, mouse wheel, or left/right mouse clicks.
  • Jump Navigation: Use ↑/↓ arrows to jump 10 files at a time, or Shift+PgDn/PgUp to jump 100 files.
  • Go to First/Last: Press Home or End.
  • Random File: Press Y to view a random file from the current folder.
  • Drag and Drop: Drop a file or folder onto the application window to open it.

Display Modes

  • Fullscreen: Toggle with the 'Fullscreen' button or by pressing F11. Double-click the media to exit fullscreen.
  • Slideshow: Press S for a sequential slideshow. Press again to speed it up.
  • Random Slideshow: Press I or F5 for a random slideshow.
  • Image Panel: Press F3 to see a thumbnail panel of all images in the folder for quick selection.
  • Always on Top: Use the checkbox to keep the window above all others.

File Operations

  • Move/Copy: Press number keys 0-9 to move (or copy) the current file to a pre-configured destination folder.
  • Rename: Press F6 or middle-click the media to rename it.
  • Delete: Press Delete to permanently delete the current file (a confirmation can be enabled/disabled).
  • Undo: Press U to undo the last move or copy operation.

Interface and Other Features

  • Folder Selection: Choose a folder using the 'Select Folder' button or by pasting a path into the folder dropdown and pressing Enter.
  • Recent Files/Folders: Quickly access recent folders from the main dropdown, or recent files from the '▼' button next to the current file name.
  • Configuration Table: Press F2 to open the settings table where you can assign destination folders to number keys (0-9) and configure other options.
  • Copy Paths: Click on the folder path or file path labels to copy them to the clipboard.
  • Image Manipulation: Press R to rotate an image clockwise, and T to rotate counter-clockwise.
  • Zoom: Use Ctrl + Mouse Wheel to zoom in and out on images. Use Alt + Mouse Wheel to reset zoom.
  • Multi-language: Switch between English and Russian interface.

3. Step-by-Step Usage Guide

  1. Initial Setup:
    • Run the application. If prompted, allow it to associate with common image file types for easier access.
    • Press F2 to open the 'Destination Folders Table'.
    • In this table, click 'Choose folder' for each key (0-9) and select a destination folder where you want to sort your files.
    • You can also set other options here, like running file operations in the background or changing the default action from 'Move' to 'Copy'.
  2. Open Your Media:
    • Click the 'Select Folder' button to browse for the folder containing the media you want to sort.
    • Alternatively, drag and drop a folder or a single media file directly onto the application window.
  3. Sort Your Files:
    • Use the navigation controls (keyboard arrows, mouse clicks/wheel) to browse through your files.
    • When you find a file you want to sort, press the number key (0-9) corresponding to the destination folder you configured. The file will be moved or copied instantly.
    • If you make a mistake, press U to undo the last action.
    • To delete a file, press the Delete key.
  4. Explore and View:
    • Use the slideshow modes (S, I, F5) for hands-free viewing.
    • Press F3 to get a quick visual overview of the entire folder in the Image Panel.
    • Use zoom and rotate functions as needed for better viewing.

Добро пожаловать в FastMediaSorter!

1. Цели приложения

FastMediaSorter предназначен для быстрой сортировки и управления большими коллекциями фотографий и видео. Основная цель — предоставить высокоэффективный рабочий процесс для пользователей, которым необходимо быстро организовывать медиафайлы по разным папкам, используя обширные сочетания клавиш для минимизации использования мыши.

2. Полный список функциональности

Просмотр и навигация по файлам

  • Следующий/Предыдущий файл: Перемещайтесь с помощью стрелок →/←, клавиш PgDn/PgUp, N/P, колеса мыши или кликов левой/правой кнопкой мыши.
  • Быстрый переход: Используйте стрелки ↑/↓ для перехода на 10 файлов вперед/назад или Shift+PgDn/PgUp для перехода на 100 файлов.
  • Перейти к первому/последнему: Нажмите Home или End.
  • Случайный файл: Нажмите Y для просмотра случайного файла из текущей папки.
  • Перетаскивание: Перетащите файл или папку в окно приложения, чтобы открыть их.

Режимы отображения

  • Полноэкранный режим: Переключите с помощью кнопки 'Fullscreen' или нажатием F11. Дважды щелкните по медиа, чтобы выйти из полноэкранного режима.
  • Слайд-шоу: Нажмите S для последовательного слайд-шоу. Нажмите еще раз, чтобы ускорить.
  • Случайное слайд-шоу: Нажмите I или F5 для случайного слайд-шоу.
  • Панель изображений: Нажмите F3, чтобы увидеть панель с миниатюрами всех изображений в папке для быстрого выбора.
  • Поверх всех окон: Используйте флажок, чтобы держать окно поверх всех остальных.

Операции с файлами

  • Перемещение/Копирование: Нажимайте цифровые клавиши 0-9, чтобы переместить (или скопировать) текущий файл в предварительно настроенную папку назначения.
  • Переименовать: Нажмите F6 или среднюю кнопку мыши на медиа, чтобы переименовать его.
  • Удалить: Нажмите Delete, чтобы безвозвратно удалить текущий файл (подтверждение можно включить/отключить).
  • Отменить: Нажмите U, чтобы отменить последнюю операцию перемещения или копирования.

Интерфейс и другие возможности

  • Выбор папки: Выберите папку с помощью кнопки 'Выбрать папку' или вставив путь в выпадающий список папок и нажав Enter.
  • Недавние файлы/папки: Быстрый доступ к недавним папкам из основного выпадающего списка или к недавним файлам с помощью кнопки '▼' рядом с именем текущего файла.
  • Таблица настроек: Нажмите F2, чтобы открыть таблицу настроек, где вы можете назначить папки назначения цифровым клавишам (0-9) и настроить другие параметры.
  • Копировать пути: Щелкните по меткам пути к папке или файлу, чтобы скопировать их в буфер обмена.
  • Манипуляции с изображением: Нажмите R для поворота изображения по часовой стрелке и T для поворота против часовой стрелки.
  • Масштабирование: Используйте Ctrl + Колесо мыши для увеличения и уменьшения масштаба изображений. Используйте Alt + Колесо мыши для сброса масштаба.
  • Многоязычность: Переключение между английским и русским интерфейсом.

3. Пошаговое руководство по использованию

  1. Начальная настройка:
    • Запустите приложение. Если будет предложено, разрешите ему ассоциироваться с распространенными типами файлов изображений для более удобного доступа.
    • Нажмите F2, чтобы открыть 'Таблицу папок-получателей'.
    • В этой таблице нажмите 'Выбрать папку' для каждой клавиши (0-9) и выберите папку назначения, куда вы хотите сортировать файлы.
    • Здесь же можно настроить другие параметры, например, выполнение файловых операций в фоновом режиме или изменение действия по умолчанию с 'Переместить' на 'Копировать'.
  2. Откройте ваши медиафайлы:
    • Нажмите кнопку 'Выбрать папку', чтобы найти папку с медиафайлами, которые вы хотите отсортировать.
    • Или перетащите папку или один медиафайл прямо в окно приложения.
  3. Сортируйте ваши файлы:
    • Используйте элементы управления навигацией (клавиши со стрелками, клики/колесо мыши) для просмотра файлов.
    • Когда вы найдете файл, который хотите отсортировать, нажмите цифровую клавишу (0-9), соответствующую настроенной вами папке назначения. Файл будет мгновенно перемещен или скопирован.
    • Если вы ошиблись, нажмите U, чтобы отменить последнее действие.
    • Чтобы удалить файл, нажмите клавишу Delete.
  4. Исследуйте и просматривайте:
    • Используйте режимы слайд-шоу (S, I, F5) для просмотра без помощи рук.
    • Нажмите F3, чтобы получить быстрый визуальный обзор всей папки на Панели изображений.
    • Используйте функции масштабирования и поворота по мере необходимости для лучшего просмотра.
